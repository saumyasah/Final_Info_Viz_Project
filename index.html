Mapping The skies : Information Visualization of Indigo Airlines

A Data Driven look at Indigo's daily & direct Domestic Flight Routes at the Busiest Airports in India.

Overview

The project provides an interesting insight into the domestic flight operations of Indigo Airlines in India. The top 25 busiest airports in India, where Indigo Airlines operates, are used as the base of the analysis. The use of Python libraries such as mpl_toolkits.basemap, matplotlib.pyplot, and pandas enables efficient data handling and visualization. The visualization is designed to showcase the hourly takeoff trends of Indigo Airlines flights in each of the 24 slots in a day. The use of a moving bar chart provides an effective way to represent the number of flights taking off in each hour. The project also leverages Flourish Studio platform to add an extra layer of visualization to the project. Overall, the project is a great example of how data visualization can be used to analyze complex datasets and provide actionable insights.
